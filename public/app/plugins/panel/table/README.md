# Table Panel -  Native Plugin

The Table Panel is **included** with Plutono.

The table panel is very flexible, supporting both multiple modes for time series as well as for table, annotation and raw JSON data. It also provides date formatting and value formatting and coloring options.

Check out the [Table Panel Showcase in the Plutono Playground](http://play.plutono.org/dashboard/db/table-panel-showcase) or read more about it here:

[https://grafana.com/docs/grafana/latest/features/panels/table_panel/](https://grafana.com/docs/grafana/latest/features/panels/table_panel/)
